createSQLXML 메서드(ISQLServerConnection)

데이터가 없는 SQLXML 개체를 만듭니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public java.sql.SQLXML createSQLXML()

반환 값

SQLXML 개체입니다.

예외

SQLServerException

설명

이 createSQLXML 메서드는 java.sql.Connection 인터페이스의 createSQLXML 메서드에 의해 지정됩니다.
