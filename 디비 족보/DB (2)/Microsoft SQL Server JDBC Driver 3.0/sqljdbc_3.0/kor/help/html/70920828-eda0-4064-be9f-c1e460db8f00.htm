setServerName 메서드(SQLServerDataSource)

SQL Server를 실행 중인 컴퓨터의 이름을 설정합니다.

구문

public void setServerName(java.lang.String serverName)

매개 변수

serverName

서버 이름이 들어 있는 String입니다.

설명

서버 이름은 SQL Server를 실행 중인 대상 컴퓨터의 호스트 이름입니다. serverName 속성이 설정되어 있지 않으면 getServerName은 기본값인 null을 반환합니다.
