position 메서드(java.sql.NClob, long)

이 NClob 개체에서 지정된 NClob 개체의 searchstr이 나타나는 문자 위치를 검색합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

long position(java.sql.NClob searchstr,
              long start)

매개 변수

searchstr

검색할 NClob 개체입니다.

start

검색을 시작할 위치이며, 첫 번째 위치는 1입니다.

반환 값

부분 문자열이 나타나는 위치이며, 부분 문자열이 없으면 -1입니다. 첫 번째 위치는 1입니다.

예외

SQLServerException

설명

이 position 메서드는 java.sql.NClob 인터페이스의 position 메서드에 의해 지정됩니다.
