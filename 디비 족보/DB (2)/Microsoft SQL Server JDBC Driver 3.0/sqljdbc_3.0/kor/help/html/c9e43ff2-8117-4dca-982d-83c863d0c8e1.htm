setResponseBuffering 메서드(SQLServerDataSource)

이 SQLServerDataSource 개체를 사용하여 만들어진 연결에 대한 응답 버퍼링 모드를 설정합니다.

구문

public void setResponseBuffering(java.lang.String value)

매개 변수

value

버퍼링 및 스트리밍 모드가 들어 있는 String 입니다. 유효한 모드는 대/소문자를 구분하지 않는 문자열 full 또는 adaptive일 수 있습니다.

설명

full 값은 런타임에 서버의 전체 결과를 읽도록 지정합니다.

adaptive 값은 필요할 때 가능한 최소한의 데이터를 버퍼링하도록 지정합니다. adaptive 값은 Microsoft SQL Server JDBC 드라이버 버전 2.0 이상의 기본 버퍼링 모드입니다.

응답 버퍼링 모드를 사용하는 방법은 선택 버퍼링 사용을 참조하십시오.
