getDateTimeOffset(java.lang.string)(SQLServerResultSet)

이 메서드는 Microsoft SQL Server JDBC 드라이버 3.0에서 추가되었습니다.

매개 변수 인덱스가 지정된 경우 지정된 열의 값을 검색하여 Java 프로그래밍 언어의 DateTimeOffset 클래스 개체로 반환합니다.

구문

public microsoft.sql.DateTimeOffset getDateTimeOffset(String columnName)

매개 변수

columnName

열 이름입니다.

반환 값

DateTimeOffset 클래스 개체입니다.

예외

SQLServerException

설명

SQLServerResultSet.updateDateTimeOffset을 사용하여 DateTimeOffset 클래스 값을 업데이트할 수 있습니다.
