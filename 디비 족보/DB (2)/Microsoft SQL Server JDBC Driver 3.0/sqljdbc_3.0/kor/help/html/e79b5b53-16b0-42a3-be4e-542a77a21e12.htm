isClosed 메서드(ISQLServerStatement)

이 SQLServerStatement 개체가 닫혔는지 여부를 나타냅니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public boolean isClosed()

반환 값

이 SQLServerStatement가 닫히면 true이고, 계속 열려 있으면 false입니다.

예외

SQLServerException

설명

이 isClosed 메서드는 java.sql.Statement 인터페이스의 isClosed 메서드에 의해 지정됩니다.
