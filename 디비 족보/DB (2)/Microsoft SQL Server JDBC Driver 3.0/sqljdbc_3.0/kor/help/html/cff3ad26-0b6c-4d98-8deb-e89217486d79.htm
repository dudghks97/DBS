getByte 메서드(int)

매개 변수 인덱스가 지정된 경우 지정된 매개 변수의 값을 검색하여 byte 값으로 반환합니다.

구문

public byte getByte(int index)

매개 변수

index

매개 변수 인덱스를 나타내는 int입니다.

반환 값

byte 값입니다.

예외

SQLServerException

설명

이 getByte 메서드는 java.sql.CallableStatement 인터페이스의 getByte 메서드에 의해 지정됩니다.
