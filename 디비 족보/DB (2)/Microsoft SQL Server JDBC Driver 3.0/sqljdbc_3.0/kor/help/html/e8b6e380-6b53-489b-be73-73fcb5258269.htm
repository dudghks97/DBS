설명

DateTimeOffset 형식은 "YYYY-MM-DD HH-MM-SS[.nnnnnnn] [+][-] HH:MM"입니다. 다음 표를 참조하십시오.

Time

"hh:mm:ss[.nnnnnnn]"만 삽입 가능

Date

"YYYY-MM-DD"만 삽입 가능

DateTime2

"YYYY-MM-DD hh:mm:ss[.nnnnnnn]"만 삽입 가능
