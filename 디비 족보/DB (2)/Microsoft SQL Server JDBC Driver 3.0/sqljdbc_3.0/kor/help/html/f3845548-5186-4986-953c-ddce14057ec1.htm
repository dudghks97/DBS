free 메서드(SQLServerClob)

이 메서드는 CLOB 개체 및 이 개체가 보유한 리소스를 해제합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public void free()

예외

SQLServerException

설명

이 free 메서드는 java.sql.Clob 인터페이스의 free 메서드에 의해 지정됩니다.
