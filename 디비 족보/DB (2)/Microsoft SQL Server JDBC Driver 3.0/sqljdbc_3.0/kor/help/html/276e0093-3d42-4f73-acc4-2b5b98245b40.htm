getXAConnection 메서드(java.lang.String, java.lang.String)

지정된 사용자 이름과 암호를 사용하여 실제 데이터베이스 연결을 설정하려고 합니다.

구문

public javax.sql.XAConnection getXAConnection(java.lang.String user,
                                              java.lang.String password)

매개 변수

user

사용자 이름이 들어 있는 String입니다.

password

암호가 들어 있는 String입니다.

반환 값

XAConnection 개체입니다.

예외

java.sql.SQLException

설명

이 getXAConnection 메서드는 javax.sql.XADataSource 인터페이스의 getXAConnection 메서드에 의해 지정됩니다.

참고: 이 메서드는 일반적으로 XA 연결 풀 구현에서 호출되며 일반적인 JDBC 응용 프로그램 코드에서는 호출되지 않습니다.
