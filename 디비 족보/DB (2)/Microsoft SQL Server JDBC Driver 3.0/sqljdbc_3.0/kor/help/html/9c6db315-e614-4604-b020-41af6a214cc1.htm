rowDeleted 메서드(ISQLServerResultSet)

행이 삭제되었는지 여부를 검색합니다.

구문

public boolean rowDeleted()

반환 값

행이 삭제되었고 삭제 내용이 검색되면 true이고, 그렇지 않으면 false입니다.

예외

SQLServerException

설명

이 rowDeleted 메서드는 java.sql.ResultSet 인터페이스의 rowDeleted 메서드에 의해 지정됩니다.

삭제된 행은 결과 집합에 가시적인 빈틈을 남겨 둘 수 있습니다. 이 메서드를 사용하면 결과 집합에서 빈틈을 검색할 수 있습니다. 반환되는 값은 이 SQLServerResultSet 개체가 삭제 내용을 검색할 수 있는지 여부에 따라 달라집니다.

참고: 검색이 정방향 및 동적 커서에 대해 일시적이기는 하지만 SQL Server에서는 업데이트 가능한 모든 커서 유형에 대해 삭제된 행을 검색합니다.
