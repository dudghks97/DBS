getUDTs 메서드(SQLServerDatabaseMetaData)

특정 스키마에 정의된 사용자 정의 형식에 대한 설명을 검색합니다.

참고: Microsoft SQL Server JDBC 드라이버에서는 현재 이 메서드가 지원되지 않습니다. 이 메서드를 사용할 경우 항상 빈 결과 집합이 반환됩니다.

구문

public java.sql.ResultSet getUDTs(java.lang.String catalog,
                                  java.lang.String schemaPattern,
                                  java.lang.String typeNamePattern,
                                  int[] types)

매개 변수

catalog

카탈로그 이름이 들어 있는 String 입니다.

schemaPattern

스키마 이름 패턴이 들어 있는 String 입니다.

typeNamePattern

형식 이름 패턴이 들어 있는 String 입니다.

types

포함할 데이터 형식이 들어 있는 int의 배열입니다. Null은 모든 형식이 포함되어야 함을 나타냅니다.

반환 값

SQLServerResultSet 개체입니다.

예외

SQLServerException

설명

이 getUDTs 메서드는 java.sql.DatabaseMetaData 인터페이스의 getUDTs 메서드에 의해 지정됩니다.
