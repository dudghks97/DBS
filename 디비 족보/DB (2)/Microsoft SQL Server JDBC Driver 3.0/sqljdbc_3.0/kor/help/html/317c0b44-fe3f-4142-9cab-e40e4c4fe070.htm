getRowIdLifetime 메서드(SQLServerDatabaseMetaData)

SQL RowId 데이터 형식이 지원되는지 여부를 나타내는 상태를 반환합니다. 이 데이터 형식이 지원되는 경우 이 메서드는 RowId 개체가 유효한 상태로 있는 수명을 반환합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public java.sql.RowIdLifetime getRowIdLifetime()

반환 값

RowIdLifetime 개체입니다.

참고: JDBC 드라이버 버전 2.0 릴리스에서 이 메서드는 java.sql.RowIdLifetime.ROWID_UNSUPPORTED 값을 반환합니다.

예외

SQLServerException

설명

이 getRowIdLifetime 메서드는 java.sql.DatabaseMetaData 인터페이스의 getRowIdLifetime 메서드에 의해 지정됩니다.
