setBytes 메서드(SQLServerBlob)

지정된 바이트 배열을 BLOB의 지정된 위치부터 쓴 다음 쓴 바이트 수를 반환합니다.

오버로드 목록

이름 설명

setBytes(long, byte[])

지정된 바이트 배열을 BLOB의 지정된 위치에서부터 쓴 다음 쓴 바이트 수를 반환합니다.

setBytes(long, byte[], int, int)

지정된 바이트 배열의 전체 또는 일부 내용을 지정된 시작 위치, 오프셋 및 길이에 따라 BLOB에 쓴 다음 쓴 바이트 수를 반환합니다.
