데이터 원본 속성 설정

데이터 원본은 Java Platform, Enterprise Edition(Java EE) 환경에서 JDBC 연결을 생성하는 기본 메커니즘입니다. 데이터 원본은 연결 속성을 Java 코드에 하드 코딩하지 않고 연결, 풀링된 연결 및 분산 연결을 제공합니다. 모든 Microsoft SQL Server JDBC 드라이버 데이터 원본은 각각 적절한 setter 및 getter 메서드를 사용하여 속성 값을 설정하거나 가져올 수 있습니다.

일반적으로 응용 프로그램 서버 및 servlet/JSP 엔진과 같은 Java EE 제품을 사용하여 데이터베이스 액세스를 위한 데이터 원본을 구성할 수 있습니다. 구성으로 속성을 속성=값 쌍으로 입력할 수 있는 경우 연결 속성 설정 항목에 소개한 속성을 지정할 수 있습니다.

SQL Server 데이터 원본에 대한 자세한 내용은 SQLServerDataSource 클래스를 참조하십시오. SQLServerDataSource 클래스를 사용하여 SQL Server 데이터베이스에 연결하는 방법에 대한 예는 데이터 원본 샘플을 참조하십시오.
