updateBytes 메서드(java.lang.String, byte[])

열 이름이 지정된 경우 지정된 열을 byte 값의 배열로 업데이트합니다.

구문

public void updateBytes(java.lang.String columnName,
                        byte[] x)

매개 변수

columnName

열 이름이 들어 있는 String입니다.

x

byte 값의 배열입니다.

예외

SQLServerException

설명

이 updateBytes 메서드는 java.sql.ResultSet 인터페이스의 updateBytes 메서드에 의해 지정됩니다.

Microsoft SQL Server JDBC 드라이버 2.0에서는 SQLServerResultSet.updateBytes를 사용하여 바이트 배열과 SQL Server 데이터 형식 date, time, datetime2 또는 datetimeoffset 간에 값을 변환할 수 있었습니다. SQL Server JDBC 드라이버 3.0에서는 이 메서드와 이러한 데이터 형식을 함께 사용하면 변환이 지원되지 않음을 나타내는 예외가 발생합니다.
