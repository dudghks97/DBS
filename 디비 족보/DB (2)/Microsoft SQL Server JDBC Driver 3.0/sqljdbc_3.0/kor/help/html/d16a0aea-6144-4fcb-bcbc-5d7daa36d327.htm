getBytes 메서드(ISQLServerResultSet)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열의 값을 검색하여 Java 프로그래밍 언어의 byte 배열로 반환합니다.

오버로드 목록

이름 설명

getBytes(int)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열 인덱스의 값을 검색하여 Java 프로그래밍 언어의 byte 배열로 반환합니다.

getBytes(java.lang.String)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열 이름의 값을 검색하여 Java 프로그래밍 언어의 byte 배열로 반환합니다.

주의

Microsoft SQL Server JDBC 드라이버 2.0에서는 SQLServerResultSet.getBytes를 사용하여 바이트 배열과 SQL Server 데이터 형식 date, time, datetime2 또는 datetimeoffset 간에 값을 변환할 수 있었습니다. SQL Server JDBC 드라이버 3.0에서는 이 메서드와 이러한 데이터 형식을 함께 사용하면 변환이 지원되지 않음을 나타내는 예외가 발생합니다.
