setClientInfo 메서드(java.lang.String, java.lang.String)

지정된 클라이언트 정보 속성의 값을 설정합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public void setClientInfo (java.lang.String name,
                           java.lang.String value)

매개 변수

name

설정할 클라이언트 정보 속성의 이름이 들어 있는 String입니다.

value

클라이언트 정보 속성에 설정할 값이 들어 있는 String입니다.

예외

SQLServerException

설명

이 setClientInfo 메서드는 java.sql.Connection 인터페이스의 setClientInfo 메서드에 의해 지정됩니다.

Microsoft SQL Server JDBC 드라이버에서는 클라이언트 정보 속성을 지원하지 않습니다. JDBC 드라이버 버전 2.0 릴리스에서 이 메서드는 속성에 대한 경고를 생성합니다. 응용 프로그램에서는 SQLServerConnection 클래스의 getWarnings 메서드를 사용하여 경고를 검색해야 합니다.
