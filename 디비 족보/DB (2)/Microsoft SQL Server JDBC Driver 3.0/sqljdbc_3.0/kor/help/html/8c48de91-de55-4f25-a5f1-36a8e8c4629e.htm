getReference 메서드(SQLServerConnectionPoolDataSource)

이 SQLServerConnectionPoolDataSource 개체에 대한 참조를 반환합니다.

구문

public javax.naming.Reference getReference()

반환 값

Reference 개체입니다.

설명

이 getReference 메서드는 javax.naming.Referenceable 인터페이스의 getReference 메서드에 의해 지정됩니다. 이 메서드는 SQLServerDataSource 클래스의 getReference 메서드를 재정의합니다.
