getCatalog 메서드(ISQLServerConnection)

이 SQLServerConnection 개체의 현재 카탈로그 이름을 검색합니다.

구문

public java.lang.String getCatalog()

반환 값

카탈로그 이름이 들어 있는 String입니다.

예외

SQLServerException

설명

이 getCatalog 메서드는 java.sql.Connection 인터페이스의 getCatalog 메서드에 의해 지정됩니다.

SQLServerConnection 개체의 현재 카탈로그 속성을 반환하거나, 이 속성이 설정되지 않은 경우 null을 반환합니다. 카탈로그 속성은 setCatalog 메서드를 사용하여 명시적으로 설정되거나, 현재 카탈로그의 TDS에 대한 환경 변경 내용을 읽을 때 암시적으로 업데이트됩니다.
