setSendStringParametersAsUnicode 메서드(SQLServerDataSource)

문자열 매개 변수를 서버에 유니코드 형식으로 보낼 수 있는지 여부를 나타내는 boolean 값을 설정합니다.

구문

public void setSendStringParametersAsUnicode(boolean sendStringParametersAsUnicode)

매개 변수

sendStringParametersAsUnicode

문자열 매개 변수를 서버에 유니코드 형식으로 보낼 수 있으면 true이고, 그렇지 않으면 false입니다.

설명

sendStringParametersAsUnicode 속성이 기본값인 true로 설정되어 있으면 문자열 매개 변수가 서버에 유니코드 형식으로 보내집니다. sendStringParametersAsUnicode 속성이 false로 설정되어 있으면 문자열 매개 변수가 서버에 유니코드 형식이 아닌 ASCII/MBCS 형식으로 보내집니다. sendStringParametersAsUnicode 속성이 설정되어 있지 않으면 getSendStringParametersAsUnicode는 기본값인 true를 반환합니다.

sendStringParametersAsUnicode 연결 속성에 대한 자세한 내용은 연결 속성 설정을 참조하십시오.
