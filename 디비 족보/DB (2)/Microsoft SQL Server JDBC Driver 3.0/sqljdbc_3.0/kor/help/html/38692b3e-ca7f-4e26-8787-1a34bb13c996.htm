valueOf 메서드(DateTimeOffset)

GMT를 기준으로 특정 오프셋의 시점을 나타내는 DateTimeOffset 개체를 만듭니다.

오버로드 목록

이름 설명

valueOf(java.sql.Timestamp, int)

지정된 java.sql.Timestamp 값과 분 단위 오프셋을 나타내는 값을 사용하여 GMT를 기준으로 특정 오프셋의 시점을 나타내는 DateTimeOffset 개체를 만듭니다.

valueOf(java.sql.Timestamp, java.util.Calendar)

지정된 java.sql.Timestamp 값과 오프셋을 나타내는 java.util.Calendar 값을 사용하여 GMT를 기준으로 특정 오프셋의 시점을 나타내는 DateTimeOffset 개체를 만듭니다.
