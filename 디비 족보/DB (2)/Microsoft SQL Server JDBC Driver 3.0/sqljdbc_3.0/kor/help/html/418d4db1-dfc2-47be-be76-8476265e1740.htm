getTimestamp 메서드(ISQLServerResultSet)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열의 값을 검색하여 Java 프로그래밍 언어의 java.sql.Timestamp 개체로 반환합니다.

오버로드 목록

이름 설명

getTimestamp(int)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열 인덱스의 값을 검색하여 Java 프로그래밍 언어의 java.sql.Timestamp 개체로 반환합니다.

getTimestamp(int, java.util.Calendar)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열 인덱스의 값을 검색하고 Calendar 개체를 사용하여 이를 Java 프로그래밍 언어의 java.sql.Timestamp 개체로 반환합니다.

이 SQLServerResultSet 개체의 현재 행에서 지정된 열 이름의 값을 검색하여 Java 프로그래밍 언어의 java.sql.Timestamp 개체로 반환합니다.

getTimestamp(java.lang.String, java.util.Calendar)
