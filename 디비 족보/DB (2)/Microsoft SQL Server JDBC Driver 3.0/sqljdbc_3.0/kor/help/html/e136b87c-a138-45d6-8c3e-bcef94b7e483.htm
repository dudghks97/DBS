샘플 JDBC 드라이버 응용 프로그램

Microsoft SQL Server JDBC 드라이버 샘플 응용 프로그램은 JDBC 드라이버의 다양한 기능을 보여 줍니다. 또한 SQL Server 데이터베이스와 함께 JDBC 드라이버를 사용할 때 참고할 수 있는 바람직한 프로그래밍 방식도 보여 줍니다.

모든 샘플 응용 프로그램은 로컬 컴퓨터에서 컴파일 및 실행된 *.java 코드 파일 형식으로 포함되며 다음 위치에 있는 여러 하위 폴더에 들어 있습니다.

<installation directory>\sqljdbc_<version>\<language>\help\samples

이 섹션의 항목에서는 샘플 응용 프로그램을 구성 및 실행하는 방법과 함께 샘플 응용 프로그램이 소개하는 내용에 대해 설명합니다.

단원 내용

항목 설명

데이터 연결 및 검색

SQL Server 데이터베이스에 연결하는 방법을 보여 줍니다. 또한 SQL Server 데이터베이스에서 데이터를 검색하는 다양한 방법도 보여 줍니다.

데이터 형식(JDBC) 사용

JDBC 드라이버 데이터 형식을 사용하여 SQL Server 데이터베이스의 데이터로 작업하는 방법을 소개합니다.

결과 집합 사용

결과 집합을 사용하여 SQL Server 데이터베이스에 포함된 데이터를 처리하는 방법을 소개합니다.

큰 데이터 작업

이러한 샘플 응용 프로그램에서는 선택 버퍼링을 사용하여 서버 커서 오버헤드 없이 SQL Server 데이터베이스에서 큰 값 데이터를 검색하는 방법에 대해 설명합니다.
