executeQuery 메서드(ISQLServerStatement)

지정된 SQL 문을 실행하고 단일 SQLServerResultSet 개체를 반환합니다.

구문

public java.sql.ResultSet executeQuery(java.lang.String sql)

매개 변수

sql

SQL 문이 들어 있는 String입니다.

반환 값

SQLServerResultSet 개체입니다.

예외

SQLServerException

설명

이 executeQuery 메서드는 java.sql.Statement 인터페이스의 executeQuery 메서드에 의해 지정됩니다.

지정된 SQL 문에서 단일 SQLServerResultSet 개체 이외의 개체를 생성하면 SQLServerException이 발생합니다.
