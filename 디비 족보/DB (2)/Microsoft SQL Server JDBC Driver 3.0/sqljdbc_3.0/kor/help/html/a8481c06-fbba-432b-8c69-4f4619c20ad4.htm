설명

SQLServerPreparedStatement는 매개 변수를 네이티브 Java 형식 및 다양한 Java 개체 형식으로 제공하는 데 사용할 수 있는 메서드를 제공합니다. SQLServerPreparedStatement는 SQL Server sp_prepare 저장 프로시저를 사용하여 문을 준비한 다음 반환된 문 핸들을 이후에 이 문을 실행할 때마다 다시 사용합니다. 이때 일반적으로 사용자가 제공하는 각기 다른 매개 변수가 사용됩니다.

SQLServerPreparedStatement는 여러 개의 준비된 문이 단일 데이터베이스 왕복으로 실행되는 일괄 처리를 지원하여 런타임 성능을 향상시킵니다.

이 클래스는 SQLServerPreparedStatement 클래스, ISQLServerPreparedStatement 인터페이스, java.sql.PreparedStatement 인터페이스 및 SQLServerStatement에서 래핑 해제를 위해 지원하는 클래스와 인터페이스에 대한 래핑 해제를 지원합니다. 자세한 내용은 래퍼 및 인터페이스를 참조하십시오.
