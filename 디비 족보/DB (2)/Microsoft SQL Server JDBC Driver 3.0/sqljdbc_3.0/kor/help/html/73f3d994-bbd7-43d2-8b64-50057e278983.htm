deletesAreDetected 메서드(SQLServerDatabaseMetaData)

SQLServerResultSet 클래스의 rowDeleted 메서드를 호출하여 표시된 행 삭제를 검색할 수 있는지 여부를 검색합니다.

구문

public boolean deletesAreDetected(int type)

매개 변수

type

결과 집합 유형을 나타내는 int 이며, java.sql.ResultSet 또는 SQLServerResultSet에 정의된 다음 값 중 하나일 수 있습니다.

java.sql.ResultSet 형식

TYPE_FORWARD_ONLY

TYPE_SCROLL_SENSITIVE

TYPE_SCROLL_INSENSITIVE

SQLServerResultSet 형식

TYPE_SS_SCROLL_STATIC

TYPE_SS_SCROLL_KEYSET

TYPE_SS_DIRECT_FORWARD_ONLY

TYPE_SS_SERVER_CURSOR_FORWARD_ONLY

TYPE_SS_SCROLL_DYNAMIC

반환 값

빈 행이 삭제된 행을 대체하면 true이고, 삭제된 행이 제거되면 false입니다.

SQL Server 데이터베이스와 Microsoft SQL Server JDBC 드라이버를 함께 사용할 경우 이 메서드는 TYPE_SS_SCROLL_KEYSET 커서에 대해 true를 반환하고 다른 모든 결과 집합 유형에 대해서는 false를 반환합니다.

예외

SQLServerException

설명

이 deletesAreDetected 메서드는 java.sql.DatabaseMetaData 인터페이스의 deletesAreDetected 메서드에 의해 지정됩니다.

참고: 검색이 정방향 및 동적 커서에 대해 일시적이기는 하지만 SQL Server에서는 업데이트 가능한 모든 커서 유형에 대해 삭제된 행을 검색합니다.
