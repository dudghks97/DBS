rollback 메서드(ISQLServerConnection)

현재 트랜잭션에서 변경한 모든 내용을 취소하고 이 SQLServerConnection 개체가 현재 보유하고 있는 데이터베이스 잠금을 해제합니다.

오버로드 목록

이름 설명

rollback()

현재 트랜잭션에서 변경한 모든 내용을 취소하고 이 SQLServerConnection 개체가 현재 보유하고 있는 데이터베이스 잠금을 해제합니다.

rollback(java.sql.Savepoint)

지정된 SQLServerSavepoint 개체가 설정된 후의 모든 변경 내용을 취소합니다.
