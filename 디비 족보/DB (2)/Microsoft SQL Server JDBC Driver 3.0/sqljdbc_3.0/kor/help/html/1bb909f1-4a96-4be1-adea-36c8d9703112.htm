updateNString 메서드(int, java.lang.String)

지정된 열 인덱스를 사용하여 지정된 열을 String 값으로 업데이트합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public void updateNString(int columnIndex,
                        java.lang.String nString)

매개 변수

columnIndex

열 인덱스를 나타내는 int 입니다.

nString

String 개체입니다.

예외

SQLServerException

설명

이 updateNString 메서드는 java.sql.ResultSet 인터페이스의 updateNString 메서드에 의해 지정됩니다.

이 메서드는 Java String을 선택된 nchar, nvarchar(max), ntext 및 xml 열에 전달합니다. 다른 데이터 형식 열에 이 메서드를 사용하면 예외가 발생합니다.
