getBigDecimal 메서드(int, int)(ISQLServerResultSet)

이 SQLServerResultSet 개체의 현재 행에서 지정된 열 인덱스의 값을 검색하고 이를 지정된 소수 자릿수를 사용하여 반환합니다.

참고: 이 메서드는 JDBC 사양에서 더 이상 사용되지 않습니다. 대신 getBigDecimal(int) 메서드를 사용해야 합니다.

구문

public java.math.BigDecimal getBigDecimal(int columnIndex,
                                          int scale)

매개 변수

columnIndex

열 인덱스를 나타내는 int입니다.

scale

소수점 이하의 자릿수를 나타내는 int입니다.

반환 값

BigDecimal 개체입니다.

예외

SQLServerException

설명

이 getBigDecimal 메서드는 java.sql.ResultSet 인터페이스의 getBigDecimal 메서드에 의해 지정됩니다.
