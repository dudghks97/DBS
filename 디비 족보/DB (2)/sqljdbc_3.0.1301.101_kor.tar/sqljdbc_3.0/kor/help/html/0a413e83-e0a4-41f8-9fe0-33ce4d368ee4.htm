setUnicodeStream 메서드(ISQLServerPreparedStatement)

지정된 매개 변수 번호를 지정된 바이트 수를 포함하는 지정된 입력 스트림으로 설정합니다.

참고: 이 메서드는 JDBC 사양에서 더 이상 사용되지 않으며 이 메서드를 호출하면 "구현되지 않음" 예외가 발생합니다.

구문

public final void setUnicodeStream(int n,
                                   java.io.InputStream x,
                                   int length)

매개 변수

n

매개 변수 번호를 나타내는 int입니다.

x

InputStream 개체입니다.

length

바이트 수입니다.

예외

SQLServerException

설명

이 setUnicodeStream 메서드는 java.sql.PreparedStatement 인터페이스의 setUnicodeStream 메서드에 의해 지정됩니다.
