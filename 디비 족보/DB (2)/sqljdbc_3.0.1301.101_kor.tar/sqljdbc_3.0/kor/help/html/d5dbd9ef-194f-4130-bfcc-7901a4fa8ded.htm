rollback 메서드(java.sql.Savepoint)

지정된 SQLServerSavepoint 개체가 설정된 후의 모든 변경 내용을 취소합니다.

구문

public void rollback(java.sql.Savepoint s)

매개 변수

s

롤백할 대상 SavePoint 개체입니다.

예외

SQLServerException

설명

이 rollBack 메서드는 java.sql.Connection 인터페이스의 rollBack 메서드에 의해 지정됩니다.

이 메서드는 자동 커밋 모드가 해제된 경우에만 사용해야 합니다.
