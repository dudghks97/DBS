getDateTimeOffset 메서드(int)

이 메서드는 Microsoft SQL Server JDBC 드라이버 3.0에서 추가되었습니다.

매개 변수 인덱스가 지정된 경우 지정된 매개 변수의 값을 검색하여 Java 프로그래밍 언어의 DateTimeOffset 클래스 개체로 반환합니다.

구문

public microsoft.sql.DateTimeOffset getDateTimeOffset(int index)

매개 변수

index

매개 변수 서수(1부터 시작)입니다.

반환 값

DateTimeOffset 클래스 개체입니다.

예외

SQLServerException

설명

SQLServerCallableStatement.setDateTimeOffset을 사용하여 DateTimeOffset 클래스 매개 변수 값을 설정할 수 있습니다.
