SQLServerBlob 생성자(SQLServerConnection, byte[])

SQLServerConnection 개체 및 byte 배열이 지정된 경우 SQLServerBlob 클래스의 새 인스턴스를 초기화합니다.

참고: JDBC 드라이버 버전 2.0에서는 이 메서드가 사용되지 않습니다. 대신 SQLServerConnection 클래스의 createBlob 메서드를 사용합니다.

구문

public SQLServerBlob(SQLServerConnection connection,
                     byte[] data)

매개 변수

connection

SQLServerConnection 개체입니다.

data

byte 배열입니다.
