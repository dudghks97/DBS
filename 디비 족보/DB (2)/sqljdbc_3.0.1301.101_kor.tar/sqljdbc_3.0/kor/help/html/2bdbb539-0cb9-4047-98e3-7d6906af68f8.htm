updateNClob 메서드(int, java.io.Reader, long)

지정된 문자 길이의 지정된 Reader 개체를 사용하여 지정된 열을 업데이트합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public void updateNClob(int columnIndex,
                        java.io.Reader reader,
                        long length)

매개 변수

columnIndex

열 인덱스를 나타내는 int 입니다.

reader

Reader 개체입니다.

length

매개 변수 데이터의 문자 수입니다.

예외

SQLServerException

설명

이 updateNClob 메서드는 java.sql.ResultSet 인터페이스의 updateNClob 메서드에 의해 지정됩니다.

이 메서드는 nvarchar(max), ntext 및 xml 열에서만 지원됩니다. 다른 데이터 형식에 이 메서드를 사용하면 예외가 발생합니다.
