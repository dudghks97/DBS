getConnection

SQLServerDataSource에서 상속되며, 이 DataSource 개체가 나타내는 데이터 원본과의 연결을 설정합니다.

getDatabaseName

SQLServerDataSource에서 상속되며, 데이터베이스 이름을 반환합니다.

getDescription

SQLServerDataSource에서 상속되며, 데이터 원본에 대한 설명을 반환합니다.

getFailoverPartner

SQLServerDataSource에서 상속되며, 데이터베이스 미러링 구성에 사용되는 장애 조치(Failover) 서버의 이름을 반환합니다.

getInstanceName

SQLServerDataSource에서 상속되며, SQL Server 인스턴스 이름을 반환합니다.

getLastUpdateCount

SQLServerDataSource에서 상속되며, lastUpdateCount 속성이 사용되는지 여부를 나타내는 boolean 값을 반환합니다.

getLockTimeout

SQLServerDataSource에서 상속되며, 데이터베이스에서 잠금 시간 초과가 보고되기 전까지 대기하는 시간(밀리초)을 나타내는 int 값을 반환합니다.

getLoginTimeout

SQLServerDataSource에서 상속되며, 이 DataSource 개체에서 연결을 시도하는 동안 대기하는 시간(초)을 반환합니다.

getLogWriter

SQLServerDataSource에서 상속되며, 모든 로깅 및 추적 메시지에 사용할 문자 출력 스트림을 반환합니다.

getPooledConnection

풀링된 연결로 사용할 수 있는 실제 데이터베이스 연결을 설정합니다.

getPortNumber

SQLServerDataSource에서 상속되며, SQL Server와의 통신에 사용되는 현재 포트 번호를 반환합니다.

getReference

이 DataSource 개체에 대한 참조를 반환합니다.

getSelectMethod

SQLServerDataSource에서 상속되며, 이 DataSource 개체를 사용하여 만들어진 모든 결과 집합에 사용되는 기본 커서 유형을 반환합니다.

getSendStringParametersAsUnicode

SQLServerDataSource에서 상속되며, 문자열 매개 변수를 유니코드 형식으로 서버에 보낼 수 있는지 여부를 나타내는 boolean 값을 반환합니다.

getServerName

SQLServerDataSource에서 상속되며, SQL Server를 실행하고 있는 컴퓨터의 이름을 반환합니다.

getURL

SQLServerDataSource에서 상속되며, 데이터 원본에 연결하는 데 사용되는 URL을 반환합니다.

getUser

SQLServerDataSource에서 상속되며, 데이터 원본에 연결하는 데 사용되는 사용자 이름을 반환합니다.

getWorkstationID

getXopenStates
