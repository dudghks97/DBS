setSendTimeAsDatetime 메서드(SQLServerDataSource)

이 메서드는 SQL Server JDBC 드라이버 3.0에서 추가되었습니다.

sendTimeAsDatetime 연결 속성의 설정을 수정합니다.

구문

public void setSendTimeAsDatetime(boolean sendTimeAsDateTime)

매개 변수

sendTimeAsDateTime

부울 값입니다. true이면 java.sql.Time 값이 SQL Server datetime 형식으로 서버에 보내집니다. false이면 java.sql.Time 값이 SQL Server time 형식으로 서버에 보내집니다.

설명

SQLServerDataSource.getSendTimeAsDatetime은 sendTimeAsDatetime 연결 속성의 설정을 반환합니다.

sendTimeAsDatetime 연결 속성에 대한 자세한 내용은 연결 속성 설정을 참조하십시오.

자세한 내용은 java.sql.Time 값을 서버에 보내는 방식 구성을 참조하십시오.
