메서드

이름 설명

free

이 메서드는 BLOB 개체 및 이 개체가 보유한 리소스를 해제합니다.

getBinaryStream

BLOB에서 데이터를 읽기 위한 입력 스트림을 반환합니다.

getBytes

BLOB 데이터를 바이트 배열로 가져옵니다.

length

BLOB 개체의 바이트 수를 반환합니다.

position

지정된 패턴과 시작 인덱스에 따라 BLOB에서 지정된 패턴이 있는 위치를 반환합니다.

setBinaryStream

BLOB 값을 쓰는 데 사용할 수 있는 스트림을 검색합니다.

setBytes

지정된 바이트 배열을 BLOB의 지정된 위치에서부터 쓴 다음 쓴 바이트 수를 반환합니다.

truncate

BLOB을 지정된 길이로 자릅니다.
