메서드

이름 설명

free

이 메서드는 NCLOB 개체 및 이 개체가 보유한 리소스를 해제합니다.

getAsciiStream

java.sql.NClob 개체로 지정된 NCLOB 값을 검색하여 ASCII 스트림으로 반환합니다.

getCharacterStream

java.sql.NClob 개체로 지정된 NCLOB 값을 검색합니다.

getSubString

java.sql.NClob 개체로 지정된 NCLOB 값에서 지정된 부분 문자열의 복사본을 검색합니다.

length

java.sql.NClob 개체로 지정된 NCLOB 값에서 문자 수를 검색합니다.

position

지정된 시작 위치에 따라 지정된 java.sql.NClob 개체 또는 해당 java.sql.NClob에 있는 부분 문자열의 문자 위치를 검색합니다.

setAsciiStream

이 java.sql.NClob 개체가 나타내는 NCLOB 값의 지정된 위치에서부터 ASCII 문자를 쓰는 데 사용할 스트림을 검색합니다.

setCharacterStream

이 java.sql.NClob 개체가 나타내는 NCLOB 값의 지정된 위치에서부터 유니코드 문자의 스트림을 쓰는 데 사용할 스트림을 검색합니다.

setString

지정된 String을 NCLOB의 지정된 위치에서부터 씁니다.

truncate

NCLOB 값을 지정된 길이로 자릅니다.
