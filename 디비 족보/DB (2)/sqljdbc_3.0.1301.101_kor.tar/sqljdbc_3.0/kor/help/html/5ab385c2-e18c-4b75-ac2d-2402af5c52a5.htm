getFetchDirection 메서드(ISQLServerResultSet)

이 SQLServerResultSet 개체의 인출 방향을 검색합니다.

구문

public int getFetchDirection()

반환 값

현재 인출 방향을 나타내는 int입니다.

예외

SQLServerException

설명

이 getFetchDirection 메서드는 java.sql.ResultSet 인터페이스의 getFetchDirection 메서드에 의해 지정됩니다.

이 메서드는 정방향 전용 커서에 대해 FETCH_FORWARD를 반환하고 다른 커서 유형에 대해서는 setFetchDirection 메서드를 호출하여 만든 마지막 설정을 반환하며, setFetchDirection 메서드를 호출한 적이 없는 경우에는 이러한 커서 유형에 대해 FETCH_UNKNOWN을 반환합니다.
