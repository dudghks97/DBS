recover 메서드(SQLServerXAResource)

리소스 관리자에서 준비된 트랜잭션 분기 목록을 가져옵니다.

구문

public javax.transaction.xa.Xid[] recover(int flags)

매개 변수

flags

XAResource.TMSTARTRSCAN 또는 XAResource.TMENDRSCAN 또는 XAResource.TMNOFLAGS 또는 XAResource.TMSTARTTRSCAN | XAResource.TMENDRSCAN 값 중 하나에 해당되는 int 값입니다.

반환 값

Xid 개체입니다.

예외

javax.transaction.xa.XAException

설명

이 recover 메서드는 javax.transaction.xa.XAResource 인터페이스의 recover 메서드에 의해 지정됩니다.

매개 변수 flag가 XAResource.TMSTARTRSCAN 또는 XAResource.TMSTARTRSCAN | XAResource.TMENDRSCAN이 아니면 복구 검색이 진행 중이어야 합니다.
