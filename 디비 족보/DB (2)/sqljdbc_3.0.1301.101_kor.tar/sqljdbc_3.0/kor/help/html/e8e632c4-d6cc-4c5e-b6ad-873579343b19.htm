getClientInfo 메서드(java.lang.String)

지정된 클라이언트 정보 속성의 값을 검색합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public java.lang.String getClientInfo (java.lang.String name)

매개 변수

name

검색할 클라이언트 정보 속성의 이름이 들어 있는 String입니다.

반환 값

클라이언트 정보 속성의 값이 들어 있는 String입니다.

예외

SQLServerException

설명

이 getClientInfo 메서드는 java.sql.Connection 인터페이스의 getClientInfo 메서드에 의해 지정됩니다.

Microsoft SQL Server JDBC 드라이버에서는 클라이언트 정보 속성을 지원하지 않습니다. 따라서 이 메서드는 null을 반환합니다.

마찬가지로 응용 프로그램에서는 SQLServerDatabaseMetaData 클래스의 getClientInfoProperties 메서드를 사용하여 드라이버에서 지원하는 클라이언트 정보 속성의 목록을 검색할 수 있습니다. getClientInfoProperties 메서드는 빈 결과 집합을 반환합니다.
