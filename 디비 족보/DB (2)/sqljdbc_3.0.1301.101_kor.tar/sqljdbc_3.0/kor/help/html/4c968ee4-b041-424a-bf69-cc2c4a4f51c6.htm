setIntegratedSecurity 메서드(SQLServerDataSource)

integratedSecurity 속성이 사용되는지 여부를 나타내는 boolean 값을 설정합니다.

구문

public void setIntegratedSecurity(boolean enable)

매개 변수

enable

integratedSecurity가 사용되면 true이고, 그렇지 않으면 false입니다.

설명

SQL Server에서 Windows 자격 증명을 사용하여 응용 프로그램 사용자를 인증하도록 하려면 "true"로 설정합니다. "true"로 설정할 경우 Microsoft SQL Server JDBC 드라이버에서는 로컬 컴퓨터 자격 증명 캐시에서 컴퓨터 또는 네트워크 로그온 시 이미 제공된 자격 증명을 검색합니다. "false"로 설정할 경우에는 사용자 이름과 암호를 제공해야 합니다.

참고: 이 속성은 Microsoft Windows 운영 체제에서만 지원됩니다.

통합 인증을 사용하는 방법에 대한 자세한 내용은 연결 URL 작성을 참조하십시오.
