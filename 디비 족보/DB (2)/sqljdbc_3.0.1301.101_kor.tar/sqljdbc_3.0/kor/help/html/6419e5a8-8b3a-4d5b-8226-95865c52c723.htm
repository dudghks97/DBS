getMoreResults 메서드(int)

이 SQLServerStatement 개체의 다음 결과로 이동하고 지정된 모드의 지침에 따라 현재 열려 있는 결과 집합 개체를 처리합니다.

구문

public final boolean getMoreResults(int mode)

매개 변수

mode

현재 열려 있는 결과 집합 개체의 처리 방법을 나타내는 int입니다. 다음 상수 중 하나여야 합니다.

CLOSE_CURRENT_RESULT

KEEP_CURRENT_RESULT(JDBC 드라이버에서는 지원되지 않음)

CLOSE_ALL_RESULTS

반환 값

반환된 결과가 결과 집합이면 true이고, 그렇지 않으면 false입니다.

예외

SQLServerException

설명

이 getMoreResults 메서드는 java.sql.Statement 인터페이스의 getMoreResults 메서드에 의해 지정됩니다.

결과가 검색되기 전에 getMoreResults 메서드를 호출할 경우 이 메서드는 mode 인수에 지정된 대로 동작하고 다음 결과로 이동합니다.

참고: JDBC 드라이버에서는 KEEP_CURRENT_RESULT 상수의 사용을 지원하지 않습니다. 이 상수를 사용하면 예외가 발생합니다.
