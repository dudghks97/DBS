setXopenStates 메서드(SQLServerDataSource)

SQL 상태를 XOPEN 규격 상태로 변환할 수 있는지 여부를 나타내는 boolean 값을 설정합니다.

구문

public void setXopenStates(boolean xopenStates)

매개 변수

xopenStates

SQL 상태를 XOPEN 규격 상태로 변환할 수 있으면 true이고, 그렇지 않으면 false입니다.

설명

xopenStates 속성이 true로 설정되어 있으면 Microsoft SQL Server JDBC 드라이버에서는 SQL 상태를 XOPEN 규격 상태로 변환합니다. 기본값은 JDBC 드라이버에서 SQL 99 상태 코드를 생성하도록 하는 false입니다. xopenStates 속성이 설정되어 있지 않으면 getXopenStates 메서드는 기본값인 false를 반환합니다.
