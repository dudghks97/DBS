getHostNameInCertificate 메서드(SQLServerDataSource)

SQL Server SSL(Secure Sockets Layer) 인증서의 유효성을 검사하는 데 사용되는 호스트 이름을 반환합니다.

구문

public java.lang.String getHostNameInCertificate()

반환 값

호스트 이름이 들어 있는 String이며, 값이 설정되어 있지 않은 경우 null입니다.

설명

호스트 이름은 통신 계층이 SSL을 사용하여 암호화된 경우 SQL Server SSL 인증서 값의 유효성을 검사하는 데 사용됩니다.

호스트 이름이 설정되어 있지 않으면 getHostNameInCertificate 메서드는 null을 반환합니다.
