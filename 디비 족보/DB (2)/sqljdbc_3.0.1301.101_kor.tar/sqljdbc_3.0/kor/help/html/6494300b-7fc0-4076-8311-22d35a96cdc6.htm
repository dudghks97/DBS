setNString 메서드(SQLServerCallableStatement)

지정된 매개 변수를 지정된 String 개체로 설정합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public final void setNString(java.lang.String parameterName, java.lang.String value)

매개 변수

parameterName

매개 변수 이름을 나타내는 String입니다.

value

String 개체입니다.

예외

SQLServerException

설명

이 메서드는 NCHAR, NVARCHAR, NTEXT 및 XML 데이터 형식에 사용해야 합니다.

이 setNString 메서드는 java.sql.CallableStatement 인터페이스의 setNString 메서드에 의해 지정됩니다.
