getEncrypt 메서드(SQLServerDataSource)

encrypt 속성이 사용되는지 여부를 나타내는 boolean 값을 반환합니다.

구문

public boolean getEncypt()

반환 값

encrypt가 사용되면 true이고, 그렇지 않으면 false입니다.

설명

encrypt 속성이 true로 설정되어 있으면 Microsoft SQL Server JDBC 드라이버에서는 서버에 인증서가 설치되어 있는 경우 SQL Server에서 클라이언트와 서버 간에 전송되는 모든 데이터에 대해 SSL 암호화를 사용하도록 지정합니다.

encrypt 속성이 지정되어 있지 않거나 false로 설정되어 있으면 드라이버에서는 SQL Server에서 SSL 암호화를 지원하도록 지정하지 않습니다. SQL Server 인스턴스가 SSL 암호화를 사용하도록 구성되어 있지 않으면 암호화 없이 연결이 설정됩니다. SSL 암호화를 사용하도록 SQL Server 인스턴스가 구성되어 있으면 Microsoft SQL Server JDBC 드라이버는 올바르게 구성된 JVM(Java Virtual Machine)에서 실행되는 경우 자동으로 SSL 암호화를 사용합니다. 그렇지 않은 경우 연결이 종료되고 드라이버에서 오류가 발생합니다. encryption 속성이 설정되어 있지 않으면 getEncrypt 메서드는 기본값인 false를 반환합니다.
