getLong 메서드(java.lang.String)

매개 변수 이름이 지정된 경우 지정된 매개 변수의 값을 검색하여 Java 프로그래밍 언어의 long으로 반환합니다.

구문

public long getLong(java.lang.String sCol)

매개 변수

sCol

매개 변수 이름이 들어 있는 String입니다.

반환 값

long 값입니다.

예외

SQLServerException

설명

이 getLong 메서드는 java.sql.CallableStatement 인터페이스의 getLong 메서드에 의해 지정됩니다.

이 메서드는 bigint, int, smallint, tinyint 및 bit와 같이 정수 값을 안전하게 반환할 수 있는 SQL Server 데이터 형식에서만 지원됩니다. 다른 데이터 형식에 이 메서드를 사용하면 예외가 발생합니다.
