unwrap 메서드(SQLServerPreparedStatement)

Microsoft SQL Server JDBC 드라이버 관련 메서드에 액세스할 수 있도록 지정된 인터페이스를 구현하는 개체를 반환합니다.

참고: 이 기능은 Microsoft SQL Server JDBC 드라이버 버전 2.0에 새로 도입된 기능입니다.

구문

public <T> T unwrap(Class<T> iface)

매개 변수

iface

인터페이스를 정의하는 T 형식의 클래스입니다.

반환 값

지정된 인터페이스를 구현하는 개체입니다.

예외

SQLServerException

설명

unwrap 메서드는 JDBC 4.0 사양에서 도입된 java.sql.Wrapper 인터페이스에 의해 정의됩니다.

응용 프로그램에서는 Microsoft SQL Server JDBC 드라이버와 관련된 JDBC API에 대한 확장에 액세스해야 할 수 있습니다. unwrap 메서드는 이 개체가 확장하는 공용 클래스에서 공급업체 확장을 노출하는 경우 이 클래스에 대한 래핑 해제를 지원합니다.

이 메서드가 호출되면 개체가 SQLServerStatement 및 SQLServerPreparedStatement 클래스로 래핑 해제됩니다.

예제 코드를 보려면 unwrap 메서드(SQLServerCallableStatement)를 참조하십시오.

자세한 내용은 래퍼 및 인터페이스를 참조하십시오.
