getBigDecimal 메서드(SQLServerCallableStatement)

지정된 매개 변수의 값을 검색하여 java.math.BigDecimal로 반환합니다.

오버로드 목록

이름 설명

getBigDecimal(int)

매개 변수 인덱스가 지정된 경우 지정된 매개 변수의 값을 검색하여 전체 자릿수를 모두 표시하는 java.math.BigDecimal로 반환합니다.

getBigDecimal(int, int)

(사용되지 않음) 매개 변수 인덱스와 소수 자릿수가 지정된 경우 지정된 매개 변수의 값을 검색하여 java.math.BigDecimal로 반환합니다.

getBigDecimal(java.lang.String)

매개 변수 이름이 지정된 경우 지정된 매개 변수의 값을 검색하여 전체 자릿수를 모두 표시하는 java.math.BigDecimal로 반환합니다.

getBigDecimal(java.lang.String, int)

(사용되지 않음) 매개 변수 이름과 소수 자릿수가 지정된 경우 지정된 매개 변수의 값을 검색하여 java.math.BigDecimal로 반환합니다.
