setFetchDirection 메서드(ISQLServerStatement)

결과 집합 행을 처리할 방향에 관한 힌트를 Microsoft SQL Server JDBC 드라이버에 제공합니다.

참고: JDBC 드라이버에서는 현재 이 메서드를 통해 제공되는 힌트가 무시됩니다.

구문

public final void setFetchDirection(int nDir)

매개 변수

nDir

다음 값 중 하나에 해당되는 행 처리 방향을 나타내는 int 입니다.

FETCH_FORWARD

FETCH_REVERSE

FETCH_UNKNOWN

예외

SQLServerException

설명

이 setFetchDirection 메서드는 java.sql.Statement 인터페이스의 setFetchDirection 메서드에 의해 지정됩니다.
