SQLServerXAResource 클래스

XA 분산 트랜잭션 관리를 위한 XAResource를 나타냅니다.

패키지: com.microsoft.sqlserver.jdbc

확장: java.lang.Object

구현: javax.transaction.xa.XAResource

구문

public class SQLServerXAResource

설명

XA 트랜잭션은 SQL Server에서 Microsoft 분산 트랜잭션 관리자(DTC)를 사용하여 구현됩니다. SQLServerXAResource 클래스는 DTC와 상호 작용하는 sqljdbc_xa.dll이라는 이름의 SQL Server 확장 dll을 호출합니다. SQLServerXAResource에서 받은 XA_START, XA_END, XA_PREPARE 등의 XA 호출은 해당하는 DTC 함수 호출에 매핑됩니다.
